ATM Machine System
A Java console-based ATM system that simulates real ATM operations with Admin and User roles.

Features
Login (Admin & User)
Deposit & Withdraw (only ₹100, ₹200, ₹500 notes)
Balance check
PIN change
Fund transfer
Mini statement (transaction history)
Admin: deposit, balance check, total transactions

Sample Login
Admin PIN: 9999
User 1: 123456 / 1234
User 2: 111111 / 1111
